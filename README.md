# AutoPrime — Landing Page de Venta de Vehículos

Landing page profesional para una concesionaria de vehículos de lujo, construida con HTML, CSS y JavaScript vanilla.

## Características

- **Diseño dark premium** con paleta dorada (#c8a84b)
- **Catálogo interactivo** con filtros por marca, tipo y precio
- **Slider de testimonios** con autoplay y navegación por puntos
- **Formulario de contacto** con validación en tiempo real
- **Animaciones** de scroll-reveal con IntersectionObserver
- **Totalmente responsive** (mobile, tablet, desktop)
- Navbar con efecto glassmorphism al hacer scroll
- Menú hamburguesa para móviles
- Botón de favoritos por tarjeta

## Estructura

```
├── index.html   # Estructura completa de la página
├── style.css    # Sistema de diseño y estilos responsivos
└── script.js    # Interactividad y lógica de UI
```

## Secciones

1. **Hero** — Encabezado principal con estadísticas y CTA
2. **Búsqueda rápida** — Filtros de marca, tipo y precio
3. **Catálogo** — 6 tarjetas de vehículos destacados
4. **Servicios** — Test drive, financiamiento, garantía, etc.
5. **Testimonios** — Slider con 3 reseñas de clientes
6. **Contacto** — Formulario + información de contacto
7. **Footer** — Links y créditos

## Uso

Abrir `index.html` directamente en el navegador. No requiere dependencias ni servidor.
